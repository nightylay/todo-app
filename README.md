# Todo List App

This project is a classic Todo List. The app helps organize everyday tasks by allowing you to mark, delete, and edit them.

## Preview

![Preview todo-list](preview.jpg 'Preview todo-list')

## Possibilities

- :pencil: Add, edit, and delete tasks
- :white_check_mark: Mark completed tasks
- :pencil2: Flexible editing:
  - Press Enter or the edit icon to save
  - Press Esc to cancel
- :mag: Real-time task search
- :mag_right: Filtering: all / active / completed
- :zero: Counting the number of remaining tasks
- :last_quarter_moon: Dark and light themes with selection saving
- :information_source: Modal window with tooltips

## Stack

- HTML5 + CSS3 (Flexbox, CSS variables, responsiveness)
- JavaScript (ES6+):
  - Modules (import/export)
  - Working with the DOM
  - localStorage for data saving

## Getting Started

To run it, you only need to clone the repository and you can run the project.
