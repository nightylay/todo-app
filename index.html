    How to use this Todo list

    Basic actions

    • Add a task: Enter text in the field and press Enter or the "+ Add" button.
    • Mark as completed: Click the checkbox to the left of the task.
    • Delete a task: Click the trash can icon .

    Editing tasks

    • Edit a task: Click the pencil icon to edit a task.
    • Save changes: To save changes, press Enter or the pencil icon .
    • You can switch between tasks: Start editing one task, then click the pencil icon of anotherâ€”the first one will be saved automatically!

    Search and filter

    • Find a task: Enter text in the search field to find the task you need.
    • Use filters:
      • All: Show all tasks
      • Active: Show only uncompleted tasks
      • Completed: Show only completed tasks.
